# MIT Flow Matching 课程中文整理

这个仓库用于整理 MIT 课程 **Generative AI With Stochastic Differential Equations** 中与 **Flow Matching**、**Diffusion Models** 相关的中文学习资料。

课程内容主要围绕以下主题展开：

- ODE / SDE 的基本概念与数值模拟
- Flow Models 与 Diffusion Models
- Flow Matching 与 Score Matching
- Guidance、Latent Space、VAE 等生成模型常见组件
- Discrete Diffusion / CTMC 及其在语言建模中的应用

## 仓库内容

当前仓库主要包含两类中文资料：

- 中文讲义整理
  - `lecture_notes_section1_zh.md` 到 `lecture_notes_section6_zh.md`
  - `lecture_notes_ch7_discrete_diffusion_zh.md`
  - `lecture_notes_appendices_zh.md`
- 中文实验 notebook
  - `lab_one_zh.ipynb`
  - `lab_two_zh.ipynb`
  - `lab_three_ipynb_zh.ipynb`

另外，仓库本地也保留了原始英文文件，便于和中文版对照阅读。

## 使用方式

- 如果你想按课程顺序阅读，可以先看各章节中文讲义。
- 如果你想边学边做，可以直接打开中文 lab notebook。
- 如果遇到公式、代码或推导不清楚的地方，可以结合原版英文材料一起对照。

## 说明

本仓库是个人学习整理版本，主要目标是帮助中文读者更顺畅地学习这门课程内容。若原课程材料后续更新，中文内容可能不会立刻同步。
